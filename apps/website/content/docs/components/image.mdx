## Usage

```tsx
import { Image } from '@fidely-ui/react'
```

```tsx
<Image />
```

### Next.js Image

Use the **asChild** prop to render the image as a child of the Image component.

```tsx
import NextImage from 'next/image'
;<Image asChild>
  <NextImage src="..." alt="..." />
</Image>
```

## Props

The **Image** component supports all native props of **HTML** image.
