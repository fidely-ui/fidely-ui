## Props

### Paper

| Prop        | Type                              | Default    | Description                                       |
| ----------- | --------------------------------- | ---------- | ------------------------------------------------- |
| `elevation` | `"0" \| "1" \| "2" \| "3" \| "4"` | `"0"`      | Controls the shadow depth of the Paper component. |
| `variant`   | `"filled" \| "outline"`           | `"filled"` | Sets the visual style of the Paper.               |
